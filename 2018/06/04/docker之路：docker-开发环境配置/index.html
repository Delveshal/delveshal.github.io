docker之路：docker 开发环境配置

这里我采用另外一种办法，使用dockercore/docker，这个镜像包含了所需要用到的依赖。镜像大概2G。

git clone

路径最好是$GOPATH/src/github.com/docker，$GOPATH可以是项目独有的。

docker pull
